我在阿里实习的57天

2021.10.21—成长的起点

至于我初入宫有多菜，我自己深知，但这些日子，只有一天出去玩了一整天剧本杀，所以也算是尽力了，我想朋友们。

这篇文章是想总结一下这些天在各方各面学到的”财富“，虽然呆的时间不长，但收获不菲。

技术汇总

  • 手写Spring IOC和AOP （IOC和AOP的思想）
  • 设计模式 （SOLID)
  • maven
  • git （太好用了）
  • json
  • mybatis
  • RPC （RPC之于微服务）
  • 环境（版本）隔离
  • 配置中心
  • 定时任务
  • 流量削峰 (消息队列)
  • DDD (充血模型和统一语言)
  • 日志、监控、核对（补偿）、兜底、降级、限流
  • 全局唯一ID （雪花算法)
  • 热部署
  • 分布式一致性 （Raft算法）
  • cola框架 （🐂)
  • Redis (缓存、分布式锁、发布订阅)
  • CodeReview
  • JOS (这不是技术，是人生态度)
  • 技术方案设计

实习前，除了JOS和git，其他最多也只是听说有这个东东。

设计模式

  • 简单工厂
  • 抽象工厂
  • 代理模式 （AOP)
  • 策略模式 (扩展点)
  • 建造者模式
  • 状态模式 （电梯）
  • facade （repository）
  • 原型设计模式
  • 命令模式 （事件）
  • 责任链 （类等级制度）
  • 适配器 （防腐层）
  • 桥梁模式 （中心）

技术方案设计

  • 基于缓存方案的库存扣减方案
  • 反向索引
  • 顺序消息
  • 扩展点
  • 流量削峰
  • 令牌桶
  • 传统数据库能否实现分布式场景下的一致性
  • 单元化的业务场景取舍
